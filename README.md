# VMS API Documentation

Dokumentasi API Vendor Management System (VMS)  
Digunakan untuk integrasi PUAS PRO.

## Getting Started
- Base URL: `https://vms.kincatonly.com/api`
- Format response: JSON
- Authentication: Bearer Token
